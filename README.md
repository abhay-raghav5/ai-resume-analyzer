# 📄 AI Resume Analyzer

An AI-powered Resume Analyzer built using Streamlit.

## 🚀 Features
- Extracts Name, Email, Phone
- Detects Skills from resume
- Identifies Projects and Experience
- Predicts best job role
- Generates ATS Score (0–100)
- Provides Smart Suggestions
- Highlights Missing Details

## 🛠️ Tech Stack
- Python
- Streamlit
- PDFPlumber
- Pytesseract (OCR)
- Plotly

## 📊 How it Works
1. Upload Resume (PDF/Image)
2. Text Extraction
3. Skill & Data Parsing
4. ATS Score Calculation
5. Suggestions & Visualization
